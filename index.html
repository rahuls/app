URL Shortener





Description:
This is a static site hosted on Github pages.
The short url is generated by making an API call to https://stark-cove-29174.herokuapp.com Endpoint.
The Endpoint is a HTTP API which accepts a URL as parameter and generates a short URL.
For instance to make a simple curl request 

        curl --header "Content-Type: application/json" \  
        --request POST \
        --data '{"lurl":"google.com"}' \
        https://stark-cove-29174.herokuapp.com
            
        #{"surl":"zgxJhY"} //Response
The JSON response has a surl(Short URL) key with a string value. Now open URL https://stark-cove-29174.herokuapp.com/{{paste your surl value here}} You will be redirected to the original URL.
The API is built using Express, uses SQLite to store the URLs and deployed on Heroku.

Code for API: https://github.com/rahuls/url-shortner.
Code for this site: https://github.com/rahuls/app.
